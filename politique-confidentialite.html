Politique de Confidentialité

La confidentialité de vos données est essentielle pour VOKA. Voici notre politique :

  • Nous ne collectons que les données strictement nécessaires au fonctionnement de la plateforme.
  • Aucune donnée personnelle n’est partagée avec des tiers sans votre consentement explicite.
  • Les fichiers déposés (documents, vidéos, audio) restent sous votre contrôle total.
  • Vous pouvez demander la suppression de vos données à tout moment via votre tableau de bord.
  • Les communications sont chiffrées via HTTPS pour garantir la sécurité.

Pour toute question, contactez-nous via le formulaire d’assistance dans votre Dashboard VOKA.
